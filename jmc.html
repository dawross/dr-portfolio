Josephson Mfg.

This page contains the case study of Josephson Mfg. website which includes the Project Overview, Tools Used and Live Links to the official product. This project is still a work in progress and has yet to be launched.

Live Link

Project Overview

The client needed a new website and wanted something simple and easy to maintain. Currently, there are only 2 sections that they needed to be dynamic, so I built a simple backend to add, edit, and delete items for those sections. There is a simple login page and once logged in, another menu will appear where they can manage those listings. The project is uploaded to AWS using Elastic Beanstalk for the backend and S3 for the front end.
